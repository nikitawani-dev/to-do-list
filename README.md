# 📝 TaskMaster - JavaScript To-Do App 

A clean and efficient task management tool designed to help users organize their daily goals. This project focuses on handling user input and saving data within the browser.

## 🚀 Live Demo
[Try the App here](https://nikitawani-dev.github.io/todo-list/)

## 🛠️ Key Features
- **Task Persistence:** Uses `localStorage` so your tasks stay saved even if you refresh the page.
- **Dynamic UI:** Add and delete tasks instantly without reloading.
- **Responsive Design:** Fully optimized for mobile and desktop screens.
- **Glassmorphism Theme:** Styled to match my professional portfolio.

## 🧠 Technical Highlights
- **DOM Manipulation:** Using JavaScript to create and remove elements dynamically.
- **Event Listeners:** Handling click and "Enter" key events for a smooth user experience.
- **JSON Handling:** Converting task arrays to strings for storage.
